## What is WebWacker?

WebWacker is a **CLI-first, Python 3 project** that recreates the nostalgic look and feel of classic terminal tools while focusing on **safe, controlled, and educational workflows**.

It is designed to help learners understand:
- How traffic behaves conceptually
- How rate, flow, and packet rhythm are modeled
- How defensive systems think about floods and spikes
- How old-school CLI tools were structured

**No real network attacks are performed.**
